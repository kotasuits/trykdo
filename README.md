# Kota Doria Wholesale — Product Catalog (GitHub Edition)

A premium textile product catalog with a serverless admin panel. This version is designed to run entirely on **GitHub for FREE**.

## 🚀 How to Launch for Free (GitHub Pages)

### 1. Create a Repository
1. Create a new repository on GitHub (e.g., `kota-catalog`).
2. Upload all the files from this folder to your repository.

### 2. Enable GitHub Pages
1. Go to your Repo **Settings** > **Pages**.
2. Under **Build and deployment** > **Source**, select `GitHub Actions`.
3. The included `.github/workflows/deploy.yml` will automatically build and host your site.

### 3. Setup Admin Access (No Server Needed)
To use the admin panel on GitHub, you need a **Personal Access Token (PAT)**:
1. Go to [GitHub Token Settings](https://github.com/settings/tokens?type=beta) (or Classic Tokens).
2. Generate a new token with **'repo'** permissions.
3. Copy this token.
4. Open your live site (e.g., `https://username.github.io/repo/loginkota.html`).
5. Select **GitHub Mode**, enter your Token and Repository Path (`username/repo`), and click Connect.

## ✨ Features
- **Zero Cost**: Hosted on GitHub Pages, data stored in repository.
- **Serverless Admin**: Upload products and images directly through the browser.
- **Responsive Design**: Shopify-style frontend for mobile and desktop.
- **WhatsApp Integration**: Customers can inquire about products instantly.

- **Frontend**: Vanilla HTML5, CSS3, JavaScript.
- **Backend (Free Mode)**: GitHub Content API.
- **Backend (Server Mode)**: PHP/Node.js compatible (legacy support included).
